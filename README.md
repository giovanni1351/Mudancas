# Mudapp

É um app que o propósito de integrar pessoas que buscam prestadores de serviço de entregas, com o foco de ajudar pessoas que estão em mudanças, que querem transportar cargas pequenas, mas não tem espaço; pessoas que tem um veículo apropriado para o transporte de cargas
